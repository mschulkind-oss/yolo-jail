# Agent Standards

This skill defines the mandatory standards for all complex tasks performed by Gemini CLI agents. These standards ensure transparency, verifiability, and coordination.

## Core Standards

See [references/core-standards.md](references/core-standards.md) for detailed instructions on:
1. **Open Questions Protocol**: Tracking and memorializing user feedback.
2. **Thinking & Synthesis**: Mandatory transcripts of reasoning.
3. **Progressive Documentation**: Recording work as it happens.
4. **Verifiable Proof**: Using tools like `showboat` for shell execution logs.
